# SecureBankCore

SecureBankCore is a backend banking system built with Java and Spring Boot. It provides REST APIs for managing users, bank accounts, and financial transactions while following clean architecture principles.

## Features

- User management
- Account management
- Deposits
- Withdrawals
- Transfers
- Transaction history

## Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git
